What do we teach?

First, we work on getting our students on the right path to becoming a "super user" in a Unix environment. Along with the basics, like tabbing between windows, students jump head first into the command line to learn about the file system. They then learn the fundamentals of JavaScript starting with variables, arrays, and so on.

Students get up to various proficiency levels depending on the program they're in. At the end of our most intensive program our students are proficient up to a level of a junior developer and we secure jobs for them in such positions. Concurrently with software development we're teaching life skills needed to be an awesome human.

Our Toolset

Cloud 9

Cloud 9 is a powerful, browser-based code editor that comes with a full Ubunto workspace. Cloud 9 lets our students get up and running quick and gives developers an easily accessible place to view a student's code.

Github

By using Github for version control we're able to see our students' progress over time and encourage them to contribute to open source projects.

Slack

Slack is our center of communication at Operation Spark. Students and our mentor developers hang out in the same channels where students can get realtime help and feedback on their code.
